# 📸 Notion 스타일 이미지 기능 가이드

## ✨ 주요 기능

### 1. 이미지 복사/붙여넣기
- **방법 1: 클립보드의 이미지 파일**
  ```
  1. 이미지를 복사 (Cmd+C / Ctrl+C)
  2. 에디터에서 붙여넣기 (Cmd+V / Ctrl+V)
  3. 자동으로 마크다운에 삽입됨
  ```

- **방법 2: Base64 이미지 데이터**
  ```
  1. 웹에서 이미지 복사 (예: 브라우저의 이미지)
  2. 에디터에서 붙여넣기
  3. Base64 형식(data:image/jpeg;base64,...)도 자동 인식
  ```

### 2. 드래그 앤 드롭 (에디터 & 미리보기 모두 지원!)
```
1. 컴퓨터에서 이미지 파일을 선택
2. 에디터 또는 미리보기 영역으로 드래그
3. 드래그 중 녹색 테두리와 "이미지를 여기에 놓으세요" 메시지 표시
4. 드롭하면 자동으로 업로드 및 삽입
```

### 3. 파일 업로드 버튼
```
✨ NEW: 미리보기 우측 상단에 "이미지 업로드" 버튼 추가
1. 미리보기 탭으로 전환
2. 우측 상단 "이미지 업로드" 버튼 클릭
3. 여러 이미지 파일 동시 선택 가능
4. 자동으로 마크다운에 삽입
```

### 4. 이미지 크기 조절 (미리보기)
```
1. 미리보기 탭으로 전환
2. 이미지에 마우스 오버
3. 좌우에 나타나는 녹색 핸들을 드래그
4. 원하는 크기로 조절 (100px ~ 1200px)
5. 크기가 마크다운 코드에 자동 반영
```

## 🔧 기술 구현 상세

### 마크다운 문법
```markdown
![image](img_1234567890_abc){width=500}
```
- `img_*`: 내부 이미지 ID (Base64로 매핑)
- `{width=500}`: 이미지 너비 (픽셀)

### 지원 형식
- **이미지 파일**: PNG, JPEG, GIF, WebP, SVG
- **Base64**: `data:image/[type];base64,[data]` 형식

### 변환 과정
1. **클립보드 이미지** → Base64 인코딩
2. **고유 ID 생성** → `img_[timestamp]_[random]`
3. **Map에 저장** → `Map<imageId, base64Data>`
4. **마크다운 삽입** → `![image](imageId){width=500}`
5. **렌더링** → Base64 데이터로 변환하여 표시

## 💻 사용 예시

### 예시 1: 스크린샷 붙여넣기
```
1. 스크린샷 캡처 (Cmd+Shift+4 on Mac)
2. 에디터에서 Cmd+V
3. ✅ 자동으로 삽입됨!
```

### 예시 2: 웹 이미지 복사
```
1. 브라우저에서 이미지 우클릭 → "이미지 복사"
2. 에디터에서 Cmd+V
3. ✅ Base64로 변환되어 삽입!
```

### 예시 3: 파일 드래그 (에디터 또는 미리보기)
```
1. Finder/탐색기에서 이미지 파일 선택
2. 에디터 또는 미리보기로 드래그
3. 녹색 테두리 + "이미지를 여기에 놓으세요" 표시
4. ✅ 자동 업로드 및 삽입!
```

### 예시 4: 미리보기에서 파일 업로드
```
1. 미리보기 탭 클릭
2. 우측 상단 "이미지 업로드" 버튼 클릭
3. 이미지 파일 선택 (여러 개 가능)
4. ✅ 한 번에 여러 이미지 삽입!
```

### 예시 5: 크기 조절
```
1. 미리보기에서 이미지 확인
2. 이미지 좌우 핸들 드래그
3. ✅ 실시간으로 크기 조절!
4. ✅ 마크다운 코드 자동 업데이트: {width=650}
```

## 🎨 UI/UX 특징

### 에디터 (왼쪽)
- **Split View**: 에디터 + 라이브 미리보기
- 이미지 ID로 표시: `![image](img_123)`
- 실시간 미리보기에 Base64 이미지 표시
- **드래그 앤 드롭 지원**: 녹색 테두리 피드백

### 미리보기 (오른쪽)
- **인터랙티브 이미지**
  - 마우스 오버 시 리사이즈 핸들 표시
  - 녹색 핸들 (좌/우)
  - 드래그 중 픽셀 표시
- **드래그 앤 드롭 지원**
  - 드래그 중 녹색 점선 테두리
  - "이미지를 여기에 놓으세요" 오버레이
  - 부드러운 애니메이션
- **이미지 업로드 버튼**
  - 우측 상단 배치
  - 다중 파일 선택 가능
- **안내 메시지**
  - 상단에 사용 팁 표시
  - 파란색 강조 배경

## 🚀 성능 최적화

### Base64 인코딩
- 클라이언트 사이드에서 처리
- 별도 서버 업로드 불필요

### 메모리 관리
- Map 자료구조로 효율적 저장
- 컴포넌트 언마운트 시 자동 정리

### 크기 제한
- 최소: 100px
- 최대: 1200px
- 반응형: max-width: 100%

## 📝 코드 구조

```

│   └── uploadedImages: Map<imageId, base64>
├── Event Handlers
│   ├── handlePaste: 붙여넣기

│   ├── getPreviewContent: 이미지 ID → Base64
│   └── Custom Image Component
└── Interactive Resize
    ├── Mouse Events
    └── Real-time Update
```

## ⚡ 단축키

- **Cmd/Ctrl + V**: 이미지 붙여넣기
- **Cmd/Ctrl + C**: 이미지 복사
- **드래그**: 이미지 크기 조절 (미리보기)

## 💡 팁

1. **대용량 이미지**: Base64는 파일 크기가 ~33% 증가하므로 적절한 크기로 리사이즈 권장
2. **다중 이미지**: 한 번에 여러 이미지 드래그 앤 드롭 가능
3. **실시간 동기화**: 에디터와 미리보기가 실시간 연동
4. **저장**: 완료 버튼 클릭 시 모든 이미지가 Base64로 마크다운에 포함되어 저장
